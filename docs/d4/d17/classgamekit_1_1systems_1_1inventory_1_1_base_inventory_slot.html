Detailed Description

template<typename TData>
class gamekit::systems::inventory::BaseInventorySlot< TData >

Base inventory slot class template.

Represents a single slot in an inventory. Each slot has an index and can hold data of type TData. Provides basic slot functionalities such as checking if the slot is empty and setting/getting the stored data.

Template Parameters
TDataType of data stored in the slot.
