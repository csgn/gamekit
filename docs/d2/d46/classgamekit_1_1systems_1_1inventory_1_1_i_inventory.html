Detailed Description

template<typename TData, typename TSettings, typename TSlot>
class gamekit::systems::inventory::IInventory< TData, TSettings, TSlot >

Interface for a generic inventory system.

Provides core functions to manage an inventory of game objects.

Template Parameters
TDataType of inventory data. Must derive from IGameObject.
TSettingsType of inventory settings. Must derive from IInventorySettings.
TSlotType of inventory slot. Must derive from IInventorySlot<TData>.
