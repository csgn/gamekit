Detailed Description

template<typename TData, typename TSettings, typename TSlot>
class gamekit::systems::inventory::BaseInventory< TData, TSettings, TSlot >

Base template class for an inventory system.

Provides standard inventory functionalities such as initialization, adding and removing items, slot queries, and capacity management.

Template Parameters
TDataType of data stored in each inventory slot.
TSettingsType of inventory settings class (must inherit from IInventorySettings).
TSlotType of inventory slot class (must implement IInventorySlot<TData> interface).
